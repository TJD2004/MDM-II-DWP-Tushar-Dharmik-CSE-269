# 📊 Case Study  
## Sorting & Filtering System – Amazon Inspired UI  

---

## 📄 Case Study Overview

This case study analyzes the sorting and filtering system of Amazon, which helps users efficiently navigate large product datasets.

A simplified version of this system is implemented using **HTML, CSS, and JavaScript**.

---

## 🎯 Objective

- Implement filtering and sorting logic  
- Improve user interaction and usability  
- Understand dynamic UI behavior  

---

## 🔍 Features Implemented

- ✅ Category Filtering  
- ✅ Price Range Filter  
- ✅ Rating-Based Filtering  
- ✅ Sorting Options:
  - Price (Low → High / High → Low)  
  - Rating  
  - Popularity  
- ✅ Real-Time Updates (No page reload)  
- ✅ Responsive Product Grid  

---

## ⚙️ Working Mechanism

- Products stored in **JavaScript array (JSON format)**  
- Filtering using `.filter()`  
- Sorting using `.sort()`  
- Dynamic DOM updates  

---

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**  

---

## 🧠 Learning Outcomes

- Understanding real-world UI patterns  
- Implementing dynamic filtering & sorting  
- Enhancing front-end interactivity  
- Improving user experience
